```
“4+1”視圖是對邏輯架構進行描述，最早由 Philippe Kruchten 提出，他在1995年的《IEEE Software》上發表了題為《The 4+1 View Model of Architecture》的論文，引起了業界的極大關注，現在是軟體設計的結構標準 - 百度百科
總的來說，只要你去做匯報/寫ppt，你總是要畫個一兩個視圖的。
4代表了4中視圖，1表示基於某一個場景，結合4中視圖進行說明。
```
